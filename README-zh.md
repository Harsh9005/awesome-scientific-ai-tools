## 文献检索与发现

- [Consensus](https://consensus.app/) - 基于AI的学术搜索引擎，从同行评审论文中提取研究发现。使用大语言模型从科学文献中综合"是/否"答案。
- [Semantic Scholar](https://www.semanticscholar.org/) - Allen研究所开发的免费AI研究工具。涵盖2亿+篇论文，提供引文上下文、TLDR摘要和影响力评分。
- [Elicit](https://elicit.com/) - AI研究助手，可查找相关论文、提取关键论断，并跨研究综合发现。
- [Connected Papers](https://www.connectedpapers.com/) - 探索学术论文关联的可视化工具。从任意种子论文构建相关论文图谱。
- [Research Rabbit](https://www.researchrabbit.ai/) - 免费的研究论文网络发现与可视化工具。监控收藏集以获取新的相关论文。
- [Scite.ai](https://scite.ai/) - 智能引文分析工具，展示论文被引用的方式——支持、对比或提及。
- [Litmaps](https://www.litmaps.com/) - 用于文献发现的动态引文地图。追踪研究脉络并发现研究空白。
- [OpenAlex](https://openalex.org/) - 全球研究体系的免费开放目录。2.5亿+学术成果，提供丰富的元数据和API访问。
- [Inciteful](https://inciteful.xyz/) - 免费工具，用于构建学术论文网络并分析引文模式。

## 引文管理

- [Zotero](https://www.zotero.org/) - 免费开源的参考文献管理器，具有浏览器集成、PDF管理以及Word/Google Docs插件。
- [Paperpile](https://paperpile.com/) - 现代参考文献管理器，具有Google Docs/Word集成和AI驱动的元数据提取。
- [Mendeley](https://www.mendeley.com/) - 具有社交网络功能的参考文献管理器，提供PDF标注和协作工具。
- [EndNote](https://endnote.com/) - 行业标准参考文献管理器，具有期刊样式数据库和Word的"边写边引"功能。
- [ReadCube Papers](https://www.papersapp.com/) - AI增强的参考文献管理器，提供智能推荐和增强型PDF阅读器。

## 研究阅读与理解

- [SciSpace](https://typeset.io/) - 论文阅读AI助手。用通俗语言解释文本、数学公式和表格。
- [Explainpaper](https://www.explainpaper.com/) - 上传论文，高亮难懂的文本，获取AI解释。
- [ChatPDF](https://www.chatpdf.com/) - 与任何PDF文档对话。对于长篇研究论文和报告尤其有用。
- [Scholarcy](https://www.scholarcy.com/) - AI摘要工具，为关键发现、方法和贡献创建知识卡片。
- [Unriddle](https://www.unriddle.ai/) - 用于理解复杂文档的AI助手。从论文中生成知识图谱。

## 写作与编辑

- [Jenni AI](https://jenni.ai/) - 专为学术写作设计的AI写作助手，支持引文和改写功能。
- [Paperpal](https://paperpal.com/) - 基于数百万已发表论文训练的AI语法和语言检查工具。由Cactus Communications开发。
- [Writefull](https://www.writefull.com/) - 基于已发表期刊文章训练的AI写作工具。已集成到Overleaf。
- [Trinka](https://www.trinka.ai/) - 面向学术和技术写作的AI语法检查工具，提供风格和语气建议。
- [QuillBot](https://quillbot.com/) - AI改写和摘要工具，具有语法检查和引文生成功能。
- [Grammarly](https://www.grammarly.com/) - 通用写作助手，具有学术语气检测和抄袭检查功能。
- [Overleaf](https://www.overleaf.com/) - 在线LaTeX编辑器，支持实时协作、模板和Writefull AI集成。
- [Typst](https://typst.app/) - 基于标记语言的现代排版系统。编译速度比LaTeX更快，错误提示更友好。

## 数据分析与可视化

- [Julius AI](https://julius.ai/) - AI数据分析师，可通过自然语言解读数据、创建可视化图表并运行统计检验。
- [Hal9](https://hal9.com/) - 基于AI的数据分析和仪表盘创建工具，支持自然语言查询。
- [Obviously AI](https://www.obviously.ai/) - 无代码机器学习平台，用于预测、分类和时间序列分析。
- [JASP](https://jasp-stats.org/) - 免费开源统计软件，同时支持频率学派和贝叶斯分析。是R/SPSS的图形界面替代品。
- [Datawrapper](https://www.datawrapper.de/) - 无需编码即可创建出版级图表、地图和表格。
- [Plotly](https://plotly.com/) - 面向Python、R和JavaScript的交互式图形库，配有Dash框架用于构建仪表盘。
- [Observable](https://observablehq.com/) - 协作数据可视化平台，具有响应式笔记本和D3.js集成。

## 科学计算

- [Wolfram Alpha](https://www.wolframalpha.com/) - 计算知识引擎，用于数学、物理、化学和工程计算。
- [COMSOL](https://www.comsol.com/) - 多物理场仿真，具有AI增强的网格划分和求解器选择。
- [MATLAB](https://www.mathworks.com/products/matlab.html) - 数值计算环境，具有AI/ML工具箱、Simulink和各领域专用包。
- [Google Colab](https://colab.research.google.com/) - 免费的Jupyter笔记本环境，提供GPU/TPU资源，用于机器学习和科学计算。
- [Weights & Biases](https://wandb.ai/) - 机器学习实验追踪、超参数调优和模型版本管理。
- [DVC](https://dvc.org/) - 面向机器学习项目的开源版本控制。与代码一起追踪数据、模型和实验。

## 图表与图形生成

- [BioRender](https://www.biorender.com/) - 科学插图工具，拥有5万+图标，覆盖生物、化学和医学图形。
- [Mermaid](https://mermaid.js.org/) - 基于Markdown文本的图表绘制工具。流程图、时序图、甘特图——全部从代码生成。
- [draw.io / diagrams.net](https://www.diagrams.net/) - 免费开源的图表绘制工具，提供科学图形库。
- [Inkscape](https://inkscape.org/) - 免费开源的矢量图形编辑器。可制作出版级SVG图形。
- [Matplotlib](https://matplotlib.org/) - Python 2D绑图库，生成出版级图形。Python科学可视化的基石。
- [Seaborn](https://seaborn.pydata.org/) - 基于Matplotlib构建的统计可视化库。为出版图形提供优美的默认样式。
- [PlotNeuralNet](https://github.com/HarisIqbal88/PlotNeuralNet) - 用于绘制神经网络架构图的LaTeX代码。
- [Excalidraw](https://excalidraw.com/) - 手绘风格白板工具。非常适合概念图和头脑风暴。

## 实验与实验室工具

- [Benchling](https://www.benchling.com/) - 生命科学研发云平台——分子生物学、CRISPR设计、库存管理。
- [LabArchives](https://www.labarchives.com/) - 电子实验室笔记本，具有版本控制、协作和机构合规功能。
- [Protocols.io](https://www.protocols.io/) - 开放获取的可重复研究方案共享平台，支持版本追踪。
- [Hive Benchmarks](https://hivebenchmark.com/) - AI驱动的实验室工作流优化和设备基准测试。

## 药物发现与化学

- [AlphaFold](https://alphafold.ebi.ac.uk/) - DeepMind的蛋白质结构预测工具。数据库中有2亿+预测结构。
- [RDKit](https://www.rdkit.org/) - 开源化学信息学工具包，用于分子分析、指纹生成和描述符计算。
- [DeepChem](https://deepchem.io/) - 面向药物发现、材料科学和量子化学的开源深度学习库。
- [ChEMBL](https://www.ebi.ac.uk/chembl/) - 开放的大规模生物活性数据库，包含200万+化合物和2000万+活性测量数据。
- [ZINC](https://zinc.docking.org/) - 用于虚拟筛选的免费商业化合物数据库。2.3亿+分子。
- [Schrödinger](https://www.schrodinger.com/) - 基于物理的药物发现和材料科学计算平台。
- [Open Babel](https://openbabel.org/) - 开源化学工具箱，用于分子文件格式转换和指纹生成。
- [PubChem](https://pubchem.ncbi.nlm.nih.gov/) - 开放化学数据库，包含1亿+化合物、生物测定和文献参考。

## 生物信息学与基因组学

- [Galaxy](https://galaxyproject.org/) - 开源Web平台，用于可访问、可重复的生物信息学分析。
- [Biopython](https://biopython.org/) - 计算分子生物学的Python工具——序列分析、结构、系统发育。
- [BLAST](https://blast.ncbi.nlm.nih.gov/) - 基本局部比对搜索工具，用于比较核苷酸或蛋白质序列。
- [Nextflow](https://www.nextflow.io/) - 工作流引擎，用于跨计算环境的可扩展、可重复的生物信息学流水线。
- [DESeq2](https://bioconductor.org/packages/DESeq2/) - R包，用于从RNA-seq计数数据进行差异基因表达分析。
- [Scanpy](https://scanpy.readthedocs.io/) - Python工具包，用于具有可扩展性能的单细胞基因表达分析。
- [IGV](https://igv.org/) - 集成基因组学查看器，用于基因组数据的交互式可视化。

## 临床试验与医学

- [ClinicalTrials.gov](https://clinicaltrials.gov/) - 美国临床研究注册平台。50万+项研究，可按疾病、干预措施和申办者搜索。
- [PubMed](https://pubmed.ncbi.nlm.nih.gov/) - 免费的生物医学文献搜索引擎。来自MEDLINE和生命科学期刊的3600万+条引文。
- [Europe PMC](https://europepmc.org/) - 开放获取的生物医学文献平台，支持全文搜索、引文网络和API访问。
- [Cochrane Library](https://www.cochranelibrary.com/) - 循证医学的金标准系统综述和荟萃分析。
- [OHDSI / OMOP](https://www.ohdsi.org/) - 基于通用数据模型的观察性健康数据分析开源工具。

## 研究用MCP服务器

MCP（模型上下文协议）服务器为AI代理提供对科学数据库的实时访问扩展。

- [PubMed MCP](https://github.com/search?q=pubmed+mcp+server) - 从PubMed/MEDLINE搜索和检索生物医学文献。
- [arXiv MCP](https://github.com/search?q=arxiv+mcp+server) - 从arXiv搜索和获取各科学领域的预印本。
- [Semantic Scholar MCP](https://github.com/search?q=semantic+scholar+mcp) - 访问Semantic Scholar API进行论文搜索、引文查询和推荐。
- [ChEMBL MCP](https://github.com/search?q=chembl+mcp+server) - 查询生物活性数据、化合物信息和靶点相互作用。
- [ClinicalTrials MCP](https://github.com/search?q=clinical+trials+mcp) - 搜索临床试验注册平台并提取研究详情。
- [bioRxiv MCP](https://github.com/search?q=biorxiv+mcp+server) - 访问bioRxiv和medRxiv的预印本。
- [CrossRef MCP](https://github.com/search?q=crossref+mcp+server) - DOI、引文和出版记录的元数据查询。
- [Google Scholar MCP](https://github.com/nicholasgriffintn/scholar-mcp-server) - 在Google Scholar中搜索学术论文和引文。

## 科学AI模型

- [AlphaFold 3](https://blog.google/technology/ai/google-deepmind-isomorphic-alphafold-3-ai-model/) - 预测蛋白质、DNA、RNA、配体的结构及其相互作用。

- [ChemBERTa](https://github.com/seyonechithrananda/bert-loves-chemistry) - 使用SMILES表示进行分子性质预测的BERT模型。
- [ESM](https://github.com/facebookresearch/esm) - 进化尺度建模——Meta AI开发的蛋白质语言模型。
- [MolBERT](https://github.com/BenevolentAI/MolBERT) - 基于BERT的分子表示学习，用于药物发现。
