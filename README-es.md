## 📚 Descubrimiento y búsqueda de literatura

- [Consensus](https://consensus.app/) - Motor de búsqueda académica impulsado por IA que extrae hallazgos de artículos científicos utilizando LLMs. 💰
- [Connected Papers](https://www.connectedpapers.com/) - Herramienta visual basada en grafos para explorar artículos académicos relacionados con un artículo semilla.
- [Dimensions](https://www.dimensions.ai/) - Plataforma de datos de investigación interconectados que vincula publicaciones, conjuntos de datos, subvenciones y ensayos clínicos.
- [Elicit](https://elicit.com/) - Asistente de investigación con IA que encuentra artículos relevantes y extrae información clave de los resúmenes.
- [Inciteful](https://inciteful.xyz/) - Herramienta gratuita que construye una red de artículos a partir de datos de citación para encontrar trabajos importantes.
- [Litmaps](https://www.litmaps.com/) - Herramienta de mapeo bibliográfico que crea mapas de citación interactivos a partir de artículos semilla.
- [OpenAlex](https://openalex.org/) - Catálogo gratuito y abierto del sistema de investigación global con una API integral.
- [Publish or Perish](https://harzing.com/resources/publish-or-perish) - Software gratuito para recuperar y analizar citas académicas.
- [Research Rabbit](https://www.researchrabbit.ai/) - Herramienta gratuita para descubrir artículos de investigación a través de redes de citación y colecciones.
- [Scite.ai](https://scite.ai/) - Análisis inteligente de citas que muestra cómo han sido citados los artículos (apoyo, contraste, mención). 💰
- [Semantic Scholar](https://www.semanticscholar.org/) - Herramienta de investigación gratuita impulsada por IA con resúmenes TLDR y análisis de citas de Allen AI.

## 📖 Lectura y anotación

- [ChatPDF](https://www.chatpdf.com/) - Interfaz de chat para hacer preguntas sobre documentos PDF.
- [Explainpaper](https://www.explainpaper.com/) - Sube un artículo y resalta texto confuso para obtener explicaciones con IA.
- [Paper Digest](https://www.paper-digest.com/) - Herramienta de IA que genera resúmenes cortos de artículos científicos.
- [Readwise Reader](https://readwise.io/read) - Aplicación de lectura mejorada con IA con resaltado, anotaciones y resúmenes GPT. 💰
- [Scholarcy](https://www.scholarcy.com/) - Resumidor de artículos impulsado por IA que crea tarjetas de resumen estructuradas. 💰
- [SciSpace](https://typeset.io/) - Copiloto de IA que explica secciones complejas, matemáticas y tablas en artículos de investigación. 💰
- [Unriddle](https://www.unriddle.ai/) - Asistente de investigación con IA que ayuda a comprender documentos complejos. 💰

## 🔬 Análisis de datos y visualización

- [BioRender](https://www.biorender.com/) - Herramienta web para crear figuras e ilustraciones científicas profesionales. 💰
- [DataRobot](https://www.datarobot.com/) - Plataforma de aprendizaje automático automatizado para construir y desplegar modelos predictivos. 💰
- [Jamovi](https://www.jamovi.org/) - Software estadístico gratuito de código abierto basado en R con una interfaz fácil de usar.
- [JASP](https://jasp-stats.org/) - Software estadístico gratuito de código abierto con análisis bayesiano y frecuentista y asistentes de IA.
- [Julius AI](https://julius.ai/) - Herramienta de análisis de datos impulsada por IA que genera código, crea visualizaciones y responde preguntas sobre tus datos.
- [Orange](https://orangedatamining.com/) - Herramienta de programación visual de código abierto para minería de datos y aprendizaje automático.
- [Tableau Public](https://public.tableau.com/) - Plataforma gratuita de visualización de datos para crear gráficos y paneles interactivos.

## ✍️ Escritura y edición

- [Authorea](https://www.authorea.com/) - Plataforma colaborativa para escribir documentos científicos con soporte de LaTeX/Markdown.
- [Grammarly](https://www.grammarly.com/) - Asistente de escritura con IA con detección de estilo académico. 💰
- [Hemingway Editor](https://hemingwayapp.com/) - Herramienta de escritura que resalta oraciones complejas y sugiere alternativas más simples.
- [Overleaf](https://www.overleaf.com/) - Editor colaborativo de LaTeX con funciones de escritura asistida por IA. 💰
- [Paperpal](https://paperpal.com/) - Asistente de escritura académica con IA entrenado con millones de artículos publicados. 💰
- [Penelope.ai](https://www.penelope.ai/) - Herramienta de IA que verifica manuscritos contra los requisitos y directrices de formato de las revistas.
- [QuillBot](https://quillbot.com/) - Herramienta de parafraseo y gramática con IA útil para la escritura académica.
- [Trinka AI](https://www.trinka.ai/) - Corrector gramatical con IA diseñado para la escritura académica y técnica.
- [Writefull](https://www.writefull.com/) - Herramienta de escritura con IA entrenada con artículos académicos publicados para retroalimentación lingüística. 💰

## 📊 Gestión de citas y referencias

- [Citation Gecko](https://citationgecko.azurewebsites.net/) - Herramienta gratuita que utiliza redes de citación para encontrar artículos relevantes.
- [Crossref](https://www.crossref.org/) - API gratuita de búsqueda de DOI y recuperación de metadatos para obras académicas.
- [EndNote](https://endnote.com/) - Software de gestión de referencias con búsqueda y formato potentes. 💰
- [Mendeley](https://www.mendeley.com/) - Gestor de referencias gratuito con lector de PDF y red social académica.
- [Paperpile](https://paperpile.com/) - Gestor de referencias con integración en Google Docs y Word. 💰
- [Reciteworks](https://reciteworks.com/) - Herramienta gratuita que verifica la consistencia entre citas en el texto y listas de referencias.
- [Zotero](https://www.zotero.org/) - Gestor de referencias gratuito y de código abierto con integración en navegadores y bibliotecas grupales.

## ➗ Ecuaciones y notación científica

- [Detexify](https://detexify.kirelabs.org/classify.html) - Dibuja un símbolo y obtén el comando LaTeX correspondiente.
- [LaTeX.js](https://latex.js.org/) - Biblioteca JavaScript que renderiza LaTeX en el navegador sin procesamiento del lado del servidor.
- [Mathpix](https://mathpix.com/) - Herramienta de IA que convierte imágenes y PDFs de matemáticas en LaTeX/MathML editable. 💰
- [MyScript](https://www.myscript.com/) - Reconocimiento de escritura manual para ecuaciones matemáticas y diagramas.

## 🎨 Generación de figuras e ilustraciones

- [Diagrams.net (draw.io)](https://www.diagrams.net/) - Herramienta de diagramación gratuita y de código abierto para diagramas de flujo, diagramas de red y más.
- [Inkscape](https://inkscape.org/) - Editor de gráficos vectoriales gratuito y de código abierto para ilustraciones científicas.
- [Lucidchart](https://www.lucidchart.com/) - Plataforma de diagramación colaborativa para crear diagramas de flujo y diagramas de sistemas. 💰
- [Mermaid](https://mermaid.js.org/) - Herramienta de diagramación y gráficos de código abierto basada en JavaScript que usa definiciones de texto.
- [PlotAPI](https://plotapi.com/) - API para crear visualizaciones de datos interactivas con calidad de publicación. 💰
- [SciDraw](https://scidraw.io/) - Repositorio gratuito de ilustraciones científicas contribuidas por investigadores.

## 🔄 Reproducibilidad y código

- [Binder](https://mybinder.org/) - Servicio gratuito que convierte repositorios Git en entornos computacionales interactivos y compartibles.
- [CML](https://cml.dev/) - CI/CD de código abierto para proyectos de aprendizaje automático por Iterative.
- [Code Ocean](https://codeocean.com/) - Plataforma de reproducibilidad computacional basada en la nube para investigadores. 💰
- [DVC](https://dvc.org/) - Sistema de control de versiones de código abierto para proyectos de aprendizaje automático y conjuntos de datos.
- [Google Colab](https://colab.research.google.com/) - Entorno de cuadernos Jupyter gratuito basado en la nube con soporte de GPU.
- [Jupyter](https://jupyter.org/) - Aplicación web de código abierto para crear y compartir documentos computacionales.
- [Papers with Code](https://paperswithcode.com/) - Recurso gratuito que vincula artículos de ML con sus implementaciones de código y conjuntos de datos.
- [Weights & Biases](https://wandb.ai/) - Seguimiento de experimentos de ML, versionado de conjuntos de datos y gestión de modelos. 💰

## 🔌 Servidores MCP para investigación

- [arXiv MCP Server](https://github.com/blazickjp/arxiv-mcp-server) - Busca y obtiene preprints de arXiv directamente dentro de agentes de codificación con IA.
- [bioRxiv/medRxiv MCP](https://github.com/cdolik/biorxiv-mcp) - Busca preprints en las bases de datos de bioRxiv y medRxiv.
- [ChEMBL MCP Server](https://github.com/jmagar/chembl-mcp) - Consulta la base de datos ChEMBL para datos de bioactividad de moléculas similares a fármacos.
- [ClinicalTrials.gov MCP](https://github.com/nv78/ClinicalTrialsMCP) - Busca y analiza datos de ensayos clínicos de ClinicalTrials.gov.
- [Google Scholar MCP](https://github.com/jdsharp/google-scholar-mcp) - Busca en Google Scholar desde agentes de IA para artículos académicos y citas.
- [OpenAlex MCP Server](https://github.com/reetp14/openalex-mcp-server) - Accede al grafo de conocimiento académico de OpenAlex a través de MCP.
- [PubMed MCP Server](https://github.com/JackKuo666/PubMed-MCP-Server) - Busca en las bases de datos de PubMed/NCBI literatura biomédica desde agentes de IA.
- [Semantic Scholar MCP](https://github.com/cyanheads/semantic-scholar-mcp-server) - Accede a la API de Semantic Scholar para búsqueda de artículos, citas e información de autores.

## 🤖 Habilidades de agentes para investigación

- [Humanizer](https://github.com/blader/humanizer) - Elimina señales de escritura generada por IA del contenido académico.
- [Multi-Agent Orchestrator](https://github.com/awslabs/multi-agent-orchestrator) - Framework para gestionar múltiples agentes de IA y enrutar solicitudes de forma inteligente.
- [Playwright Skill](https://github.com/nicepkg/playwright-mcp) - Habilidad de automatización de navegadores para web scraping y recopilación de datos.
- [Trail of Bits Security Skills](https://github.com/trailofbits/mcp-skills) - Habilidades enfocadas en seguridad para auditoría de código y análisis de vulnerabilidades.

## 🧬 Herramientas específicas por dominio

### Descubrimiento de fármacos y farmacia

- [ADMETlab](https://admetmesh.scbdd.com/) - Plataforma integral de predicción de propiedades ADMET utilizando aprendizaje automático.
- [AlphaFold](https://alphafold.ebi.ac.uk/) - Sistema de IA de DeepMind para predecir estructuras 3D de proteínas a partir de secuencias de aminoácidos.
- [ChEMBL](https://www.ebi.ac.uk/chembl/) - Base de datos abierta de moléculas bioactivas similares a fármacos con datos de unión, funcionales y ADMET.
- [DeepChem](https://deepchem.io/) - Biblioteca de Python de código abierto para aprendizaje profundo en descubrimiento de fármacos y química.
- [pkCSM](https://biosig.lab.uq.edu.au/pkcsm/) - Herramienta para predecir propiedades farmacocinéticas y de toxicidad usando firmas basadas en grafos.
- [SwissADME](http://www.swissadme.ch/) - Herramienta gratuita para calcular propiedades fisicoquímicas, farmacocinéticas y de similitud con fármacos.

### Bioinformática y genómica

- [BLAST](https://blast.ncbi.nlm.nih.gov/) - Herramienta gratuita para comparar secuencias de nucleótidos o proteínas contra bases de datos.
- [DAVID](https://david.ncifcrf.gov/) - Base de datos para anotación, visualización y descubrimiento integrado para análisis de enriquecimiento génico.
- [Ensembl](https://www.ensembl.org/) - Navegador de genomas para genomas de vertebrados con anotación génica y genómica comparativa.
- [Galaxy](https://usegalaxy.org/) - Plataforma web de código abierto para biología computacional accesible y reproducible.
- [UniProt](https://www.uniprot.org/) - Base de datos integral de secuencias proteicas e información funcional.

### Química

- [Avogadro](https://avogadro.cc/) - Editor y herramienta de visualización molecular gratuito y de código abierto.
- [ChemDraw](https://revvitysignals.com/products/research/chemdraw) - Herramienta estándar de la industria para dibujar estructuras químicas. 💰
- [PubChem](https://pubchem.ncbi.nlm.nih.gov/) - Base de datos abierta de química con información sobre estructuras químicas, propiedades y bioactividades.
- [RDKit](https://www.rdkit.org/) - Kit de herramientas de quimioinformática de código abierto para informática química y aprendizaje automático.

### Investigación clínica

- [OpenClinica](https://www.openclinica.com/) - Sistema de captura electrónica de datos de código abierto para ensayos clínicos.
- [REDCap](https://www.project-redcap.org/) - Aplicación web segura para construir y gestionar encuestas en línea y bases de datos para investigación clínica.
- [TrialBulletin](https://trialbulletin.com/) - Plataforma de inteligencia de ensayos clínicos impulsada por IA.

## 💰 Redacción de subvenciones y financiamiento

- [GrantForward](https://www.grantforward.com/) - Motor de búsqueda de subvenciones impulsado por IA que conecta investigadores con oportunidades de financiamiento. 💰
- [NIH Reporter](https://reporter.nih.gov/) - Base de datos gratuita y consultable de proyectos de investigación financiados por los NIH y sus resultados.
- [NSF Award Search](https://www.nsf.gov/awardsearch/) - Herramienta de búsqueda gratuita para proyectos y premios financiados por la NSF.
- [Pivot-RP](https://pivot.proquest.com/) - Base de datos integral de oportunidades de financiamiento con emparejamiento de investigadores. 💰
- [Research Professional](https://www.researchprofessional.com/) - Alertas de oportunidades de financiamiento y noticias sobre políticas de investigación. 💰

## 📢 Presentación y comunicación

- [Beautiful.ai](https://www.beautiful.ai/) - Herramienta de presentación con IA con formato inteligente y sugerencias de diseño. 💰
- [Canva](https://www.canva.com/) - Plataforma de diseño con funciones de IA para crear pósters, infografías y presentaciones.
- [Gamma](https://gamma.app/) - Creador de presentaciones impulsado por IA con plantillas profesionales.
- [Lumen5](https://lumen5.com/) - Plataforma de creación de videos con IA para convertir investigaciones en resúmenes de video atractivos. 💰
- [SlidesAI](https://www.slidesai.io/) - Herramienta de IA que crea diapositivas de presentación a partir de texto.

## 📤 Envío y publicación

- [DOAJ](https://doaj.org/) - Directorio de Revistas de Acceso Abierto: una lista curada de revistas de acceso abierto de calidad.
- [Editage Journal Selector](https://www.editage.com/journal-selector) - Recomendación de revistas impulsada por IA basada en el resumen del manuscrito.
- [Jane](https://jane.biosemantics.org/) - Herramienta gratuita que encuentra las mejores revistas para tu manuscrito basándose en el título/resumen.
- [JournalFinder (Elsevier)](https://journalfinder.elsevier.com/) - Herramienta gratuita para encontrar revistas que coincidan con el alcance de tu artículo.
- [Sherpa Romeo](https://v2.sherpa.ac.uk/romeo/) - Base de datos de políticas de acceso abierto de editoriales y permisos de autoarchivo.
- [Think Check Submit](https://thinkchecksubmit.org/) - Herramienta de lista de verificación para ayudar a los investigadores a identificar revistas confiables.
