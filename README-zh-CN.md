## 📚 文献发现与检索

- [Consensus](https://consensus.app/) - 基于大语言模型从科学论文中提取研究发现的 AI 学术搜索引擎。 💰
- [Connected Papers](https://www.connectedpapers.com/) - 基于可视化图谱探索与种子论文相关学术文献的工具。
- [Dimensions](https://www.dimensions.ai/) - 关联出版物、数据集、基金和临床试验的综合研究数据平台。
- [Elicit](https://elicit.com/) - 能够找到相关论文并从摘要中提取关键信息的 AI 研究助手。
- [Inciteful](https://inciteful.xyz/) - 通过引文数据构建论文网络以发现重要文献的免费工具。
- [Litmaps](https://www.litmaps.com/) - 从种子论文创建交互式引文地图的文献映射工具。
- [OpenAlex](https://openalex.org/) - 提供全面 API 的免费开放全球科研系统目录。
- [Publish or Perish](https://harzing.com/resources/publish-or-perish) - 用于检索和分析学术引文的免费软件。
- [Research Rabbit](https://www.researchrabbit.ai/) - 通过引文网络和文献集合发现研究论文的免费工具。
- [Scite.ai](https://scite.ai/) - 智能引文分析工具，展示论文被引用的方式（支持、对比、提及）。 💰
- [Semantic Scholar](https://www.semanticscholar.org/) - 由 Allen AI 开发的免费 AI 研究工具，提供 TLDR 摘要和引文分析。

## 📖 阅读与标注

- [ChatPDF](https://www.chatpdf.com/) - 可以对 PDF 文档进行提问的对话界面。
- [Explainpaper](https://www.explainpaper.com/) - 上传论文并高亮难懂的文字，获取 AI 解释。
- [Paper Digest](https://www.paper-digest.com/) - 为科学论文生成简短摘要的 AI 工具。
- [Readwise Reader](https://readwise.io/read) - 支持高亮标注和 GPT 摘要的 AI 增强阅读应用。 💰
- [Scholarcy](https://www.scholarcy.com/) - 创建结构化摘要卡片的 AI 文章总结工具。 💰
- [SciSpace](https://typeset.io/) - 帮助解释研究论文中复杂段落、数学公式和表格的 AI 副驾驶。 💰
- [Unriddle](https://www.unriddle.ai/) - 帮助理解复杂文档的 AI 研究助手。 💰

## 🔬 数据分析与可视化

- [BioRender](https://www.biorender.com/) - 用于创建专业科学图表和插图的网络工具。 💰
- [DataRobot](https://www.datarobot.com/) - 用于构建和部署预测模型的自动化机器学习平台。 💰
- [Jamovi](https://www.jamovi.org/) - 基于 R 构建、界面友好的免费开源统计软件。
- [JASP](https://jasp-stats.org/) - 支持贝叶斯和频率学分析及 AI 辅助的免费开源统计软件。
- [Julius AI](https://julius.ai/) - 能够生成代码、创建可视化图表并回答数据问题的 AI 数据分析工具。
- [Orange](https://orangedatamining.com/) - 用于数据挖掘和机器学习的开源可视化编程工具。
- [Tableau Public](https://public.tableau.com/) - 用于创建交互式图表和仪表板的免费数据可视化平台。

## ✍️ 写作与编辑

- [Authorea](https://www.authorea.com/) - 支持 LaTeX/Markdown 的科学文档协作写作平台。
- [Grammarly](https://www.grammarly.com/) - 支持学术风格检测的 AI 写作助手。 💰
- [Hemingway Editor](https://hemingwayapp.com/) - 高亮复杂句子并建议更简洁表达的写作工具。
- [Overleaf](https://www.overleaf.com/) - 带有 AI 辅助写作功能的协作 LaTeX 编辑器。 💰
- [Paperpal](https://paperpal.com/) - 基于数百万已发表文章训练的 AI 学术写作助手。 💰
- [Penelope.ai](https://www.penelope.ai/) - 根据期刊要求和格式指南检查稿件的 AI 工具。
- [QuillBot](https://quillbot.com/) - 适用于学术写作的 AI 改写和语法工具。
- [Trinka AI](https://www.trinka.ai/) - 专为学术和技术写作设计的 AI 语法检查工具。
- [Writefull](https://www.writefull.com/) - 基于已发表学术论文训练的 AI 写作语言反馈工具。 💰

## 📊 引用与文献管理

- [Citation Gecko](https://citationgecko.azurewebsites.net/) - 利用引文网络查找相关论文的免费工具。
- [Crossref](https://www.crossref.org/) - 用于学术作品的免费 DOI 查询和元数据检索 API。
- [EndNote](https://endnote.com/) - 具有强大搜索和格式化功能的文献管理软件。 💰
- [Mendeley](https://www.mendeley.com/) - 带有 PDF 阅读器和学术社交网络的免费文献管理工具。
- [Paperpile](https://paperpile.com/) - 支持 Google Docs 和 Word 集成的文献管理工具。 💰
- [Reciteworks](https://reciteworks.com/) - 检查正文引用与参考文献列表一致性的免费工具。
- [Zotero](https://www.zotero.org/) - 支持浏览器集成和群组文库的免费开源文献管理工具。

## ➗ 公式与科学符号

- [Detexify](https://detexify.kirelabs.org/classify.html) - 手绘符号即可获取对应 LaTeX 命令。
- [LaTeX.js](https://latex.js.org/) - 无需服务器端处理即可在浏览器中渲染 LaTeX 的 JavaScript 库。
- [Mathpix](https://mathpix.com/) - 将数学公式图片和 PDF 转换为可编辑 LaTeX/MathML 的 AI 工具。 💰
- [MyScript](https://www.myscript.com/) - 数学公式和图表的手写识别工具。

## 🎨 图表与插图生成

- [Diagrams.net (draw.io)](https://www.diagrams.net/) - 用于流程图、网络图等的免费开源绘图工具。
- [Inkscape](https://inkscape.org/) - 用于科学插图的免费开源矢量图形编辑器。
- [Lucidchart](https://www.lucidchart.com/) - 用于创建流程图和系统图的协作绘图平台。 💰
- [Mermaid](https://mermaid.js.org/) - 使用文本定义的开源 JavaScript 图表和流程图工具。
- [PlotAPI](https://plotapi.com/) - 用于创建交互式出版级数据可视化的 API。 💰
- [SciDraw](https://scidraw.io/) - 由研究人员贡献的免费科学插图资源库。

## 🔄 可重复性与代码

- [Binder](https://mybinder.org/) - 将 Git 仓库转化为可交互、可共享计算环境的免费服务。
- [CML](https://cml.dev/) - 由 Iterative 开发的面向机器学习项目的开源 CI/CD 工具。
- [Code Ocean](https://codeocean.com/) - 面向研究人员的云端计算可重复性平台。 💰
- [DVC](https://dvc.org/) - 面向机器学习项目和数据集的开源版本控制系统。
- [Google Colab](https://colab.research.google.com/) - 支持 GPU 的免费云端 Jupyter 笔记本环境。
- [Jupyter](https://jupyter.org/) - 用于创建和共享计算文档的开源 Web 应用程序。
- [Papers with Code](https://paperswithcode.com/) - 将机器学习论文与其代码实现和数据集关联的免费资源。
- [Weights & Biases](https://wandb.ai/) - 机器学习实验跟踪、数据集版本管理和模型管理平台。 💰

## 🔌 科研 MCP 服务器

- [arXiv MCP Server](https://github.com/blazickjp/arxiv-mcp-server) - 在 AI 编程代理中直接搜索和获取 arXiv 预印本。
- [bioRxiv/medRxiv MCP](https://github.com/cdolik/biorxiv-mcp) - 从 bioRxiv 和 medRxiv 数据库搜索预印本。
- [ChEMBL MCP Server](https://github.com/jmagar/chembl-mcp) - 查询 ChEMBL 数据库获取类药分子的生物活性数据。
- [ClinicalTrials.gov MCP](https://github.com/nv78/ClinicalTrialsMCP) - 从 ClinicalTrials.gov 搜索和分析临床试验数据。
- [Google Scholar MCP](https://github.com/jdsharp/google-scholar-mcp) - 从 AI 代理中搜索 Google Scholar 学术论文和引文。
- [OpenAlex MCP Server](https://github.com/reetp14/openalex-mcp-server) - 通过 MCP 访问 OpenAlex 学术知识图谱。
- [PubMed MCP Server](https://github.com/JackKuo666/PubMed-MCP-Server) - 从 AI 代理中搜索 PubMed/NCBI 生物医学文献数据库。
- [Semantic Scholar MCP](https://github.com/cyanheads/semantic-scholar-mcp-server) - 访问 Semantic Scholar API 进行论文搜索、引文分析和作者信息查询。

## 🧬 领域专用工具

### 药物发现与制药

- [ADMETlab](https://admetmesh.scbdd.com/) - 基于机器学习的综合 ADMET 性质预测平台。
- [AlphaFold](https://alphafold.ebi.ac.uk/) - DeepMind 开发的从氨基酸序列预测蛋白质三维结构的 AI 系统。
- [ChEMBL](https://www.ebi.ac.uk/chembl/) - 包含结合、功能和 ADMET 数据的生物活性类药分子开放数据库。
- [DeepChem](https://deepchem.io/) - 用于药物发现和化学领域深度学习的开源 Python 库。
- [pkCSM](https://biosig.lab.uq.edu.au/pkcsm/) - 基于图签名预测药代动力学和毒性性质的工具。
- [SwissADME](http://www.swissadme.ch/) - 计算理化性质、药代动力学和类药性的免费工具。

### 生物信息学与基因组学

- [BLAST](https://blast.ncbi.nlm.nih.gov/) - 将核苷酸或蛋白质序列与数据库进行比对的免费工具。
- [DAVID](https://david.ncifcrf.gov/) - 用于基因富集分析的注释、可视化和集成发现数据库。
- [Ensembl](https://www.ensembl.org/) - 提供基因注释和比较基因组学的脊椎动物基因组浏览器。
- [Galaxy](https://usegalaxy.org/) - 用于可访问、可重复计算生物学的开源网络平台。
- [UniProt](https://www.uniprot.org/) - 综合蛋白质序列和功能信息数据库。

### 化学

- [Avogadro](https://avogadro.cc/) - 免费开源的分子编辑器和可视化工具。
- [ChemDraw](https://revvitysignals.com/products/research/chemdraw) - 行业标准的化学结构绘图工具。 💰
- [PubChem](https://pubchem.ncbi.nlm.nih.gov/) - 包含化学结构、性质和生物活性信息的开放化学数据库。
- [RDKit](https://www.rdkit.org/) - 用于化学信息学和机器学习的开源化学信息学工具包。

### 临床研究

- [OpenClinica](https://www.openclinica.com/) - 用于临床试验的开源电子数据采集系统。
- [REDCap](https://www.project-redcap.org/) - 用于构建和管理临床研究在线调查和数据库的安全 Web 应用程序。
- [TrialBulletin](https://trialbulletin.com/) - AI 驱动的临床试验情报平台。

## 💰 基金申请与资助

- [GrantForward](https://www.grantforward.com/) - 将研究人员与资助机会进行匹配的 AI 基金搜索引擎。 💰
- [NIH Reporter](https://reporter.nih.gov/) - 可搜索的 NIH 资助研究项目和成果免费数据库。
- [NSF Award Search](https://www.nsf.gov/awardsearch/) - NSF 资助项目和奖项的免费搜索工具。
- [Pivot-RP](https://pivot.proquest.com/) - 带有学者匹配功能的综合资助机会数据库。 💰
- [Research Professional](https://www.researchprofessional.com/) - 资助机会提醒和研究政策新闻。 💰

## 📢 演示与学术交流

- [Beautiful.ai](https://www.beautiful.ai/) - 具有智能排版和设计建议的 AI 演示工具。 💰
- [Canva](https://www.canva.com/) - 带有 AI 功能的设计平台，用于创建海报、信息图和演示文稿。
- [Gamma](https://gamma.app/) - 带有专业模板的 AI 演示文稿构建工具。

- [SlidesAI](https://www.slidesai.io/) - 从文本输入创建演示幻灯片的 AI 工具。

## 📤 投稿与出版

- [DOAJ](https://doaj.org/) - 开放获取期刊目录——精选的优质开放获取期刊列表。
- [Editage Journal Selector](https://www.editage.com/journal-selector) - 基于稿件摘要的 AI 期刊推荐工具。
